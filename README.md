一份**互動式自我介紹履歷**，基於 React + Vite 製作。支援**深色/淺色主題切換**、**中英文切換**、**動畫進場效果**，並可直接部署到 GitHub Pages。

---


## 技術棧 / Framework

- ⚛️ [React](https://react.dev/) 18
- ⚡️ [Vite](https://vitejs.dev/)
- 🎨 CSS3 + 自訂 RWD
- 💡 [Framer Motion](https://www.framer.com/motion/)（動畫效果）
- 🌐 GitHub Pages 自動部署
